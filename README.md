Playable in-browser [here](https://jb-frog.itch.io/dont-pop-me-now).

## To Edit Levels:
- Key Press = Modes:
    - B = Place Bubbles
    - O = Place Obstacles
    - G = Place Goals
    - N = No Edit Mode / Return to Game
- Editing:
    - Left Click and drag to place objects
    - Right Click to delete objects
    - Num keys to load level
